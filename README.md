# 返利群发助手微信小程序

这是一个原生微信小程序项目，包含微信号/微信群管理、返利链接抓取分类、商品筛选和发送任务生成。

## 功能范围

- 管理多个微信号和每个微信号下的微信群。
- 通过云函数 `fetchDeals` 抓取 `http://www.msmds.com.cn/about-us.html` 页面中的返利/优惠链接。
- 按电商网站和商品类型分类展示返利链接。
- 按评分筛选商品并生成微信群发送文案。
- 点击“一键发送”会复制文案并记录任务，再引导用户手动切换到微信群发送。

## 重要限制

微信小程序不能直接控制个人微信号或微信群自动发消息，也不能作为群控机器人使用。因此本项目没有实现后台自动群发，而是采用“生成文案 + 复制 + 用户确认发送”的方式。

## 使用方式

1. 用微信开发者工具打开当前目录。
2. 在 `project.config.json` 中替换自己的小程序 `appid`。
3. 开通云开发后，上传并部署 `cloudfunctions/fetchDeals`。
4. 在“账号”页添加微信号和微信群。
5. 在“商品”页刷新抓取返利链接并筛选。
6. 在“发送”页选择商品，生成文案并复制发送。

## 后续可扩展

- 接入云数据库，把账号、群、商品和任务从本地缓存迁移到云端。
- 增加商品黑名单、关键词白名单、佣金比例等筛选条件。
- 增加任务状态流转，例如待发送、已复制、已发送、失败。
